# vite-cname-plugin

A simple plugin to generate a CNAME file

## Installation

```
yarn add vite-cname-plugin
```

or

```
npm install vite-cname-plugin
```

## Define this plugin

```js
import viteCNAMEPlugin from 'vite-cname-plugin'

{
    plugins: [
        viteCNAMEPlugin({
            domain: 'your.domain'
        })
    ]
}
```
